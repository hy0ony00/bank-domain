# Banking Domain Study

금융권 프로젝트 수행에 필요한 은행 업무와 시스템 도메인을 학습하고 정리한 저장소입니다.

단순한 금융 용어 정리보다는 업무 흐름과 실제 금융 시스템에서의 처리 방식을 이해하는 것을 목표로 합니다.

## 학습 내용

| 구분 | 내용 |
|---|---|
| 여신일반 | 여신의 개념, 분류, 심사 및 업무 절차 |
| 여신상품 | 가계·기업·신용·담보대출 등 여신상품 |
| 여신담보 및 관리 | 담보, 채권보전, 연체 및 사후관리 |
| 자금세탁방지 | CDD, EDD, STR, CTR |
| 금융 시스템 | 계정계, 정보계, 대외계, 원장, 배치 |
